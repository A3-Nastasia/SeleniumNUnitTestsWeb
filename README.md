# A colletion of Selenium + NUnit Tests

## Create a project with `dotnet` sdk
Create a project:
```
$ dotnet new console -o <project_name>
```

To add packages:
```
$ dotnet add package <package_name>
```

List of packages:
```
$ dotnet list package
```
